TrueTurist

Explore flight routes and date grids with a small web UI backed by the fli library. You are on the static project site; scans run wherever you host the Python API.

Source

https://github.com/mwtre/trueturist

Run locally

Clone, then from flight-explorer (repo layout already has fli next to it):

git clone https://github.com/mwtre/trueturist.git
cd trueturist/flight-explorer
make setup
make run

Open http://127.0.0.1:8765. Check /api/health if something looks off.

GitHub Pages only serves this HTML. The trip finder uses FastAPI and server-sent events, so run uvicorn locally (or deploy that stack elsewhere); Pages cannot run that backend.
