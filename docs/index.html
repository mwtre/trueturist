TrueTurist

Flight discovery tooling: a local web UI scans routes and calendars (powered by the fli library). This page is the static site on GitHub Pages; the live scanner runs on your machine or any host where you start the API.

Repository

Source code: github.com/mwtre/trueturist

Run the app locally

From the flight-explorer folder (with fli as a sibling):

make setup
make run

Then open http://127.0.0.1:8765 — health check: /api/health.

Why GitHub Pages shows this page

Pages only serves static files. The trip finder UI talks to FastAPI streaming endpoints; those must be running separately (local uvicorn or your own server). This site documents how to do that.

Deployed from the docs/ folder via GitHub Actions → GitHub Pages.
